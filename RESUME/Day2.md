# My Day2 Resume
## Section1: Dictionary
I learned:
- How dictionaries are been created
- _Syntaxe_: **dictionary_name = {"key1":"value1","key2":"value2",...}**
- That variables in a dictionary are called **keys**
- How to change the value of a key and add a new key to the dictionary
- How to delete a key using the _.pop()_ method and delete a whole dictionary using the _del_ function.
- _Syntaxe_: **del dictionary_name**

## Section2: if-else
I learned:
- That **if** is used in cases conditions have to be tested
- That for multiple conditions the **elif(elif=else if)** statement can be used
- It's possible to have nested conditional statements i.e. a conditional statement in another one.

## Section3: Loops
I learned:
- Comparism operators: ==(equal to), =!(differnt from), <(less than), >(greater than), <=(less than or equal to), >=(greater than or equal to)
- Logical operators: **and** and **or**
- That loops can be nested
### while loop
- It's use to repeat a block of instructions as long as the initial condition is true
- It's possible to prematurely stop the loop using the _break_ key word
- It's possible to continue the looping after a certain condition is reached using the _continue_ key word.

### For loop
- It's use for iterations over a sequence
- It's mostly used accompagnied by the **in** key word

## Section4: Functions
I learned:
- How to create a function using the **def** key word
- _Syntaxe_: **def function_name():**
- That to return a value from a function, the **return** key word is used
- That parameters can be added to a function
- _Syntaxe_: **def function_name(a,b):**; where a and b are the parameters
- That a function being created can be reused later by calling it i.e. calling a function is writing the function name where needed in the code and including parameters(if the initial function had parameters).
